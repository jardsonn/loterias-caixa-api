Informações

Loterias CAIXA API, como o próprio nome já diz, é uma API REST que foi construída usando o framework Spring Boot e a biblioteca Jsoup para extrair e os dados do site da CAIXA. Como a base dessa API é um Web Scraping, qualquer alteração na estrutura do site da CAIXA acarretará em um erro.

Para mais informações, acesse a documentação completa no Swagger-UI.
